# P2P Encrypted File Sharing (Java)

A secure Peer-to-Peer file transfer system built using Core Java.

### Features:
- **P2P Architecture:** No central server required.
- **AES Encryption:** Files are encrypted using AES-128 before transmission.
- **Buffered Streams:** Efficient handling of file data.

### How to Run:
1. Run `Receiver.java` first.
2. Run `Sender.java` to start the transfer.
